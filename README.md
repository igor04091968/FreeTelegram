# FreeTelegram

Open-source toolkit for устойчивого доступа к коммуникационным сервисам при ограничениях сети.

## Что это
FreeTelegram — это набор клиентских и инфраструктурных инструментов, которые упрощают безопасное подключение и помогают поддерживать доступность сервиса при сетевых ограничениях.

## Статус
Idea / pre-MVP.
