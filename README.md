# FreeTelegram

Open-source toolkit for устойчивого доступа к коммуникационным сервисам при ограничениях сети.

## Что это
FreeTelegram — это набор клиентских и инфраструктурных инструментов, которые упрощают безопасное подключение и помогают поддерживать доступность сервиса при сетевых ограничениях.

## Идея
Проект задуман как самостоятельно развивающийся и адаптирующийся к меняющимся условиям блокировок. Мы используем ИИ для анализа сети, логов и телеметрии, чтобы автоматически обновлять маршруты, подстраивать параметры транспорта и выпускать обновления, не требуя ручных действий от пользователя.
